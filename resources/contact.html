Contact & Testing

Use the following curl commands to test the server's functionality, security, and concurrency.

GET Requests

Test file downloads using curl. The `-O` flag saves the output to a file.

curl -O http://localhost:8080/sample.txt curl -O http://localhost:8080/logo.png curl -O http://localhost:8080/photo.jpg

POST Requests

Send JSON data to test the POST endpoint for file uploads.

curl -X POST http://localhost:8080/upload \ -H "Content-Type: application/json" \ -d '{"name": "test", "message": "Hello World"}'

Security Testing

These requests test security features and should return error responses.

curl http://localhost:8080/../etc/passwd curl -H "Host: evil.com" http://localhost:8080/ curl -X PUT http://localhost:8080/test

Concurrency Testing

This script tests multiple simultaneous connections to the server.

for i in {1..10}; do curl http://localhost:8080/large_image.png & done wait
